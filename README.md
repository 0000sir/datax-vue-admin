# datax-vue-admin

[后端项目github地址](https://github.com/huzekang/springboot-datax.git)

该项目由 [vue-admin-template](https://github.com/PanJiaChen/vue-admin-template/) 修改而来

## 说明

### 修改启动的端口 

找到 `vue.config.js` 修改 `proxy` 里的属性即可

## Build Setup

运行 

```
npm install
```

打包

```
npm run build:prod
```
